# CppOpenSSL - An [OpenSSL](https://www.openssl.org/) wrapper for C++

# Features
* Wrap OpenSSL 1.1.1
* Cross-platform: works on Windows, Linux, MacOS, Android, iOS, etc
* Usual-algorithm: support aes, base64, hmac, md5, pem, rsa
